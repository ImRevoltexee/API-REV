# API-REV

Open-source REST API platform built with Node.js and Express.

## Features

* REST API endpoints
* JSON responses
* Easy integration
* Open-source
* Developer-friendly

## Installation

```bash
git clone https://github.com/ImRevoltexee/API-REV.git

cd API-REV

npm install

npm start
```

## Usage

Start the server:

```bash
npm start
```

Default URL:

```text
http://localhost:3000
```

## Example Response

```json
{
  "status": true,
  "creator": "Revolt",
  "message": "API is running successfully"
}
```
